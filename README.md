# 2. Часть 1 — Модульное тестирование функции arccos(x)

## 2.1 Теоретическая основа

Функция `arccos(x)` реализована через разложение:

\[
arccos(x) = \frac{\pi}{2} - arcsin(x)
\]

где:

\[
arcsin(x)=\sum_{n=0}^{\infty} \frac{(2n)!}{4^n(n!)^2(2n+1)}x^{2n+1}
\]

Особенность: ряд Тейлора для `arcsin(x)` медленно сходится при \( x \to \pm1 \).

Для повышения численной устойчивости были применены:

- Явная обработка граничных значений:
  - `arccos(1) = 0`
  - `arccos(-1) = π`
- Для значений \( x \approx +1 \):
  \[
  arccos(x)=2\arcsin\left(\sqrt{\frac{1-x}{2}}\right)
  \]
- Для значений \( x \approx -1 \):
  \[
  arccos(x)=\pi-arccos(-x)
  \]

Это устраняет проблемы сходимости на границе области определения.
